<h1> Setting local notification in android - react native </h1>

<p>
Simple project to setup local notification with the help of react-native-push-notification
</p>

<h2>
How to install
</h2>

<p>
Install the packages using
</p>
<code>
npm i
</code>
<p>
Run using
</p>
<code>
react-native run-android
</code>
